# Mòdul 0227 - Serveis de xarxa

### RA1- Serveis de configuració dinàmica (DHCP) 
### RA2- Serveis de resolució de noms (DNS)
### RA3- Serveis de transferència de fitxers (FTP)
### RA4- Servidors de correu electrònic 
### RA5- Servidors web
### RA6- Sistemes d'accés remot (SSH)
### RA7- Desplegament de xarxes sense fils
### RA8- Accés des de xarxes locals a públiques
